This is an industry practicum (capstone) project worked in collaboration with Kearney for a real client. They are a leading enthusiatic auto part manufacturer in the USA. We've been engaged to evaluate the efficiency of the current network by evaluating the placement of the existing Logistic Centers (LC). If the existing network is inefficient w.r.t to cost or service objectives, we propose alternatieve cities. The problem is approached using assignment problem a flavor of facility location in order to optimize the flow of demand from one point to the other. We used solvers from Google-OR tools and Gurobi to optimize the mathematical model.
